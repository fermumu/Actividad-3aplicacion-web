Acerca de Navidad

La Navidad se celebra en todo el mundo, tanto como fiesta religiosa como secular. Para los cristianos, este día anual se celebra para recordar el nacimiento de Jesús. Se entregan regalos, se envían tarjetas navideñas, se cantan villancicos y se asiste a servicios religiosos especiales.
